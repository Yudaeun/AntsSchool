주요 기능

📚 경제 학습 (20챕터)

입문·초급·중급 3단계. 챕터당 카드 5~7장 + 퀴즈 3문제. 5분 이내 완료 가능한 구조로 설계.

📰 경제 뉴스

연합뉴스·매일경제·한국경제·머니투데이 RSS 파싱. 무한 스크롤, 북마크, 기사 본문 스크래핑(Jsoup).

🎯 퀴즈 시스템

데일리 퀴즈, 오답노트 자동 저장, 연속 정답 콤보 보너스 XP.

🏆 게임화 (Gamification)

레벨·경험치·스트릭(연속 출석)·배지 시스템. Google 계정 연동으로 기기 변경 시에도 데이터 유지.

💹 실시간 시장 정보

홈 화면에 주요 지수(코스피·나스닥·환율 등) 실시간 표시.

📊 내 기록 통계

레벨·경험치 진행도, 학습 캘린더(스트릭), 배지 컬렉션, 퀴즈 정답률 통계.
