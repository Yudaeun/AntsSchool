## 🎯 기획 의도

주식·ETF·금리 등 경제 개념이 낯선 개미 투자자들이 **듀오링고처럼 쉽고 재미있게** 경제를 배울 수 있는 앱을 목표로 기획했습니다.

- 챕터 하나를 **5분 이내**에 완료할 수 있도록 설계
- 게임화 요소(레벨·배지·스트릭)로 학습 지속성 유도
- Google 계정 연동으로 기기가 바뀌어도 학습 데이터 유지

## 🛠 기술 스택

### Android (클라이언트)

| 분류 | 기술 |
|------|------|
| **언어** | Kotlin |
| **UI** | Jetpack Compose + Material3 |
| **아키텍처** | MVVM (ViewModel + StateFlow) |
| **네트워크** | Retrofit2 + OkHttp |
| **인증** | Firebase Authentication (Google 로그인) |
| **광고** | Google AdMob |
| **빌드** | Gradle Kotlin DSL |

### Server (백엔드)

| 분류 | 기술 |
|------|------|
| **언어** | Kotlin |
| **프레임워크** | Spring Boot 3.3 |
| **ORM** | Spring Data JPA |
| **DB** | PostgreSQL (Railway 클라우드) |
| **스크래핑** | Jsoup (RSS 파싱 + 기사 본문) |
| **배포** | Railway |

## ✨ 주요 기능

### 📚 경제 학습 (20챕터)
- 입문 6챕터 · 초급 8챕터 · 중급 6챕터
- 챕터당 카드 5~7장 + 퀴즈 3문제
- 챕터 완료 시 배지 획득 · 잠금/해금 구조

### 📰 경제 뉴스
- 연합뉴스 · 매일경제 · 한국경제 · 머니투데이 RSS 파싱
- 무한 스크롤 · 뉴스사별 필터 칩
- 기사 본문 스크래핑(Jsoup) · 북마크 저장

### 🎯 퀴즈 시스템
- 데일리 퀴즈 · 오답노트 자동 저장
- 연속 정답 콤보 보너스 XP

### 🏆 게임화 (Gamification)
- 레벨 · 경험치 · 스트릭(연속 출석) · 배지 시스템
- 학습 캘린더 · 퀴즈 정답률 통계

### 💹 홈 화면
- 오늘의 경제 한 줄 · 주요 지수 실시간 표시
- 학습 진도 카드 · 최신 뉴스 미리보기

## 💡 개발 회고

### 잘된 점
- Jetpack Compose로 모든 UI를 선언형으로 구성해 상태 관리가 명확했음
- MVVM + StateFlow 패턴으로 UI와 비즈니스 로직을 분리
- Spring Boot 서버와 Railway 배포를 2주 안에 완성

### 어려웠던 점
- 뉴스 RSS 파싱 + 본문 스크래핑 시 언론사마다 HTML 구조가 달라 처리 필요
- AdMob 중복 의존성(Duplicate class) 오류 해결
- Firebase UID 기반 서버 사용자 관리 설계

### 향후 계획
- AI 퀴즈 (Claude Haiku API 연동) 구현
- Play Store 정식 출시
- 프리미엄 구독 (광고 제거 · AI 퀴즈 무제한)
